# Animal Collector

## Overview
Animal Collector is a first-person exploration game where the player walks through nature after setting up a camping spot and photographs animals they encounter in the environment.

The main mechanic revolves around using an in-game camera to capture photos of wildlife. Every picture taken is displayed on the camera screen and saved inside the project folder, allowing the player to build a collection of animal photographs while exploring the world.

This project was created as a Unity prototype focusing on exploration mechanics and camera-based gameplay systems.

---

## Features

- First-person exploration gameplay
- In-game camera system for taking photos
- Photos displayed directly on the camera screen
- Automatic saving of captured photos to a project folder
- Simple wildlife observation and collection mechanic
- Natural environment designed for exploration

---

## Gameplay

The player explores a natural environment using a first-person controller. While walking through the environment, animals can be discovered and photographed using the in-game camera.

When a photo is taken:

1. The camera captures the current scene.
2. The image is rendered on the in-game camera screen.
3. The photo is stored inside the project folder.

This allows players to build their own collection of animal photographs as they explore.
